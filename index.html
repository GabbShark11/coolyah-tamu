Soft Skill

Apa itu Soft Skill ?

Soft Skill ?

Soft skill adalah kumpulan kemampuan interpersonal yang memungkinkan seseorang untuk berinteraksi dengan orang lain secara efektif. Contohnya seperti berikut :

  • Komunikasi
  • Kerjasama Tim
  • Berpikir Kritis & Pemecahan Masalah
  • Networking
  • Manajemen Diri

Mengapa kita perlu Soft Skill ?

Alasannya adalah karena Soft Skill sangat dibutuhkan dalam dunia kerja.

Setiap dunia kerja akan menghadapi berbagai jenis masalah, Organisasi/perusahaan membutuhkan orang-orang yang mampu berpikir kritis dan mampu memecahkan sebuah masalah dengan baik.

Hard Skill

Apa itu Hard Skill ?

Hard Skill ?

Hard Skill adalah keterampilan teknis yang dibutuhkan untuk melaksanakan suatu pekerjaan. Hard skill dapat dipelajari dan ditingkatkan melalui: Latihan, Pengulangan, Pendidikan formal, Pelatihan, Pengalaman langsung. Contohnya seperti berikut :

  • Programming
  • Database Management
  • Networking
  • Cyber Security
  • Cloud Computing
  • UI/UX

Mengapa kita membutuhkan Hard Skill ?

Alasannya adalah karena Hard skill sangat diperlukan untuk menyelesaikan tugas-tugas tertentu dalam bidang formal maupun non formal.

Hard skill merupakan kemampuan teknis yang spesifik dan dapat diukur secara objektif.
